Features

Text + Voice Complaint Input

Submit issue details using text and a simulated voice input interface.

QR Scan Simulation

Auto-fill bus ID instantly with one click to mimic QR code scan.

Location Capture

Capture GPS coordinates from browser geolocation for better traceability.

Media Proof Upload

Attach image/video evidence to support complaint authenticity.

NLP Category Simulation

Auto-detect issue category from keywords like delay, overcrowding, misconduct.

Status Tracking + Admin Panel

Track complaint progress and let admins update status in a single dashboard.

How It Works

1 Passenger submits complaint with details and evidence.
2 System auto-categorizes complaint based on text keywords.
3 Admin views complaint and updates resolution status.
4 Passenger tracks status using complaint ID.
